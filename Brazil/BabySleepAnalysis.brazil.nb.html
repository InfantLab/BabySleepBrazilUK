Sleeping times

Next we want to work out how long the babies slept each night based on a giong to sleep time and a wake time. Need to be a little careful because mostly babies go to bed one calendar day and wake the next but sometimes they go to bed after midnight so need to handle both conditions.


lengthsleep<-function(sleep, wake){
  #how long did baby sleep
  sleeptime <- wake - sleep
  if (sleeptime < 0){
    #sleep time less that zero we add 24hours
    sleeptime<- sleeptime +  24*60*60  #1day
  }
  return (sleeptime)
}
